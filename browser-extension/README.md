# 豆包梦幻皮肤浏览器扩展

适用于网页版豆包的 Chrome / Microsoft Edge 扩展。扩展只在 `doubao.com` 页面运行，背景图片和设置保存在本机浏览器中。

## 安装到 Chrome

1. 下载并解压整个项目。
2. 在地址栏打开 `chrome://extensions/`。
3. 打开右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本项目中的 `browser-extension` 文件夹。
6. 打开 [网页版豆包](https://www.doubao.com/chat/)，建议把“豆包梦幻皮肤”固定到工具栏。

## 安装到 Microsoft Edge

1. 下载并解压整个项目。
2. 在地址栏打开 `edge://extensions/`。
3. 打开左侧的“开发人员模式”。
4. 点击“加载解压缩的扩展”。
5. 选择本项目中的 `browser-extension` 文件夹。
6. 打开 [网页版豆包](https://www.doubao.com/chat/) 即可。

安装后请保留项目文件夹。浏览器会从这个目录加载扩展，移动或删除后需要重新选择。

## 使用方法

1. 点击浏览器工具栏中的“豆包梦幻皮肤”。
2. 点击“自定义背景、文字与完整主题”。
3. 先选择“暖霞玻璃、午夜霓虹、樱花梦境、海盐微风”之一，也可以直接逐项自定义。
4. 选择 JPG、PNG 或 WebP 图片（最大 16 MB），调整组件和文字参数，点击“保存并应用”。

已打开的豆包页面会自动更新，无需刷新。弹窗中的开关可以临时暂停皮肤并恢复网页原外观。

## 支持的设置

- 背景图片、背景焦点
- 背景亮度、饱和度
- 玻璃模糊强度
- 左侧栏颜色与透明度
- 主区域颜色与透明度
- 输入框颜色与透明度
- 四套完整主题预设
- 柔和玻璃、清晰描边、沉浸实体三种组件风格
- 导航选中态、主按钮、链接、输入焦点和滚动条强调色
- 功能卡片、用户气泡、豆包气泡的颜色与透明度
- 全局组件圆角与阴影强度
- 智能深色 / 浅色文字适配
- 固定深色、固定浅色、自定义主要文字与次要文字颜色

智能模式会在本机缩小并采样背景图片，然后分别结合侧栏、主区域和输入框的玻璃颜色与透明度计算对比度。设置页会实时显示三个区域的选色结果和平均对比度；整个过程不会上传图片。

## 权限与隐私

- `storage`：保存主题设置和本地背景图片。
- `unlimitedStorage`：允许保存不超过 16 MB 的背景图片，避免浏览器默认存储额度不足。
- `doubao.com` 网站权限：仅用于在网页版豆包中添加背景和玻璃样式。

扩展不会上传背景图片，不读取或发送聊天内容，不包含远程代码，也不会在其他网站运行。卸载扩展后，浏览器保存的主题数据会一并删除。

## 更新扩展

从 GitHub 获取新版并覆盖项目文件后，在扩展管理页面点击本扩展的“重新加载”按钮即可。
